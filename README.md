# Sales Analysis Project

## Overview
This project analyses sales data to identify trends, product performance, and business opportunities.

## Business Problem
Businesses need to understand which products generate the most profit and how sales change over time.

## Tools Used
- Excel
- Tableau

## Process
- Cleaned and organised the dataset
- Analysed sales and profit trends
- Created visual dashboards

## Key Findings
- Certain products generate significantly higher profit
- Sales trends vary across time periods

## Business Recommendation
Focus on high-performing products and optimise low-performing areas to increase overall profitability.
